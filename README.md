# Acertijo lógica jarras de agua
Es una app desarrollada con Java mediante la esctructura de datos de árboles para reolver el siguiente acertijo:

Tenemos dos jarras o botellas una que puede medir 5 litros y otra 3 litros. ¿Cómo podemos conseguir tener 4 litros justos?

# JDK 1.7
Esta app fue desarrollada Java SE Development Kit 1.7
